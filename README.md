# Raspberry_Petcare
一种基于树莓派的家庭宠物投喂装置，主要构成部件如下：
**摄像头云台(2)、树莓派3B开发板(3)、散热扇(4)、第一舵机(5)、第二舵机(6)、摄像头(7)，音响(8)压力传感器(9)、粮盆(10)、粮瓶台(11)、
粮瓶(12)、粮入口(13)、粮瓶挡板(14)、第三舵机(15)、投水机(16)、水盆(17)以及水瓶(18)。**
